Nguyễn Thị Hạnh An                                                                                                                                                      Chu Tiến Dũng                                                                                                                                                           Trần Thiện Sáng                                                                                                                                                        -Dự án Restaurant Management System là hệ thống quản lý nhà hàng hỗ trợ nhân viên và quản trị viên thực hiện toàn bộ quy trình đặt món, thanh toán qua VNPAY, quản lý thực đơn và theo dõi đơn hàng theo thời gian thực                                                                                                                       Ứng dụng được phát triển bằng C# (.NET 8.0) và ASP.NET Core Web API, sử dụng Entity Framework Core để thao tác dữ liệu trên cơ sở dữ liệu SQL Server, kết hợp VNPAY SDK để xử lý thanh toán trực tuyến và SignalR để cập nhật trạng thái đơn hàng tức thời                                                                                      Cấu trúc chính của dự án gồm các thư mục: Controllers chứa các API chính như Food, Order, Vnpay, Account; Models định nghĩa các thực thể dữ liệu như Food, User, Order; Repositories xử lý thao tác CRUD                                                                                                                                        Để cài đặt và chạy hệ thống, cần có môi trường .NET SDK 8.0, SQL Server 2019+, và Visual Studio 2022 hoặc VS Code. Sau khi import file cơ sở dữ liệu restaurant_db.sql trong thư mục /Database vào SQL Server, mở file appsettings.json để chỉnh lại chuỗi kết nối "DefaultConnection" phù hợp với tên máy chủ thực tế                         Tài khoản demo gồm Admin (tranvinh@gmail.com / 12345), Khách hàng (tranthiensang.tvlnd@gmail.com / 12345)
